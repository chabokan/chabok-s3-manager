    Build برای معماری‌های مختلف
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


🖥️  ویندوز x86_64:
───────────────────────────────────────────────────

    npm run build:win

📁 خروجی:
    dist/Chabokan S3 Manager Setup 1.0.0.exe


🐧 لینوکس x86_64 (AMD64):
───────────────────────────────────────────────────

    npm run build:linux

📁 خروجی:
    dist/chabokan-s3-manager_1.0.0_amd64.deb
    dist/Chabokan S3 Manager-1.0.0.AppImage


🍎 macOS (Intel/M1):
───────────────────────────────────────────────────

    npm run build:mac

📁 خروجی:
    dist/Chabokan S3 Manager-1.0.0.dmg


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 تنظیمات فعلی:

✅ Windows: x64 (64-bit Intel/AMD)
✅ Linux:   x64 (64-bit Intel/AMD)
✅ macOS:   universal (Intel + M1/M2)

💡 نکته:

اگر می‌خواهید برای ARM64 هم Build بزنید:

    # ویرایش package.json
    "arch": ["x64", "arm64"]

اما معمولاً x64 برای اکثر کاربران کافی است!
